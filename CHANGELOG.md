# Changelog

All notable changes to this project should be documented in this file.

## [0.1.0] - 2026-04-10

### Added

- initial `zkc` compiler prototype
- arithmetic DSL with `public`, `private`, `let`, `constrain`, and `expose`
- typed HIR stage and backend abstraction
- lowering to a wire-based arithmetic circuit IR
- witness execution and constraint checking
- examples, tests, CI, and project documentation
